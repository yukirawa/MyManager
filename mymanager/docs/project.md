# MyManager 開発資料 (v1.1)

## 技術選定
* **言語**: Dart
* **UI**: Flutter
* **DB: Isar Database** (ローカル保存・オフライン対応)

## なぜ作るのか
自己管理のため。今まで色々自己管理を試みたもののどれも失敗。最終手段としてソフトウェアに頼ることにします。
